## 🚀 رفع الموقع على GitHub Pages

1. ارفعي ملف `index.html` في الـ repository
2. اذهبي إلى **Settings → Pages**
3. اختاري **Deploy from a branch → main → / (root)**
4. انتظري دقيقتين ثم افتحي:

```
https://USERNAME.github.io/sama-crochet/
```
